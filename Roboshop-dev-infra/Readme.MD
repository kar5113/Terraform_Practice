This is the readme file for Roboshop Dev Infra repository.


## Repository Structure
The repository is structured into multiple directories, each representing a specific component of the Roboshop Dev Infra setup. Below is a brief description of each directory:
- `00-VPC`: Contains Terraform configurations for setting up the Virtual Private Cloud (VPC) for Roboshop.
- `10-SG`: Contains Terraform configurations for setting up Security Groups for Roboshop.
- `20-bastion`: Contains Terraform configurations for setting up the Bastion host for Roboshop.
